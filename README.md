### 🔬 What I'm Working On

- **[Nano-LLaDA](https://github.com/JayCheng113/Nano-LLaDA)** — A ~30M parameter discrete diffusion language model combining autoregressive and diffusion-based generation. Implements LLaDA 1.0 & 2.0 techniques including iterative remasking sampling and block curriculum learning.

- **[skill-retrieval-mcp](https://github.com/JayCheng113/skill-retrieval-mcp)** — MCP server that gives AI agents instant access to 89K+ expert skill guides via semantic search. Supports Claude Code, Cursor, Gemini CLI, and Codex.

- **[skill-finder](https://github.com/JayCheng113/skill-finder)** — Zero-setup companion to skill-retrieval-mcp. Drop in a single file, no MCP server needed.

- **[EZVibe-Light](https://github.com/JayCheng113/EZVibe-Light)** — Lightweight terminal UI for managing multiple Claude Code sessions across projects. 43KB bundle, zero native deps.

- **[PaperFarm](https://github.com/JayCheng113/PaperFarm)** — Let AI agents run experiments in any repo while you sleep.
